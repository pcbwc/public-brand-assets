# Public Brand Assets

A lightweight file server for brand assets, images, and resources.

## Purpose
This repository serves static assets that can be referenced via public URLs, primarily for:
- Salesforce authentication page images
- Brand logos and graphics
- Public-facing resources

## Usage
Files in this repository are accessible via GitHub's raw content URL:
```
https://raw.githubusercontent.com/pcbwc/public-brand-assets/main/<path-to-file>
```

## Structure
```
/images/          - Image assets
/logos/           - Brand logos
/salesforce/      - Salesforce-specific assets
```

## Adding Assets
1. Add your files to the appropriate directory
2. Commit and push to main branch
3. Access via the raw GitHub URL
